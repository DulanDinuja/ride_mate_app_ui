# Assets Directory

## Required Images

### google_icon.png
- Download the official Google "G" logo from: https://developers.google.com/identity/branding-guidelines
- Place it in this directory as `google_icon.png`
- Recommended size: 24x24 pixels or higher (with transparent background)

Alternatively, you can use the fallback icon (Icons.g_mobiledata) that's already implemented in the code.
